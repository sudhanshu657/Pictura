📸 Imagify – AI Image Generator (MERN Stack + ClipDrop API)
Imagify is a full-stack AI SaaS application that converts user text prompts into high-quality images using the ClipDrop API. Built with the MERN stack, it offers user authentication, a credit-based system, and fast, responsive image generation.

🚀 Features
Convert text prompts into AI-generated images using ClipDrop API.

User authentication with JWT and secure password hashing.

MongoDB-based user system to manage accounts and credit balances.

Initial 5 free credits per user; additional credits can be purchased.

Responsive UI built with React.js and Tailwind CSS.

Real-time feedback and notifications via Toastify.

Fast image rendering in < 5 seconds per request.


🛠️ Tech Stack
Frontend: React.js, Tailwind CSS, Axios, React Router, Toastify

Backend: Node.js, Express.js, MongoDB, Mongoose, JWT, bcrypt

AI API: ClipDrop (Stability AI)

Deployment: Vercel (frontend), Render (backend), MongoDB Atlas (database)
